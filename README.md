# Quiz Game (Python)

## Description

This Quiz Game is a simple Python console application that allows users to answer multiple-choice questions and receive a final score based on their correct answers. It is designed to practice basic Python programming concepts in an interactive way.

## Features

- Multiple-choice quiz questions
- User-friendly console interface
- Instant answer validation
- Score calculation
- Displays final result
- Beginner-friendly implementation

## Technologies Used

- Python
- IDLE / VS Code / Any Python Interpreter

## How to Run

1. Download or clone the repository.
2. Open the project in a Python IDE or editor.
3. Run the `Quiz_game.py` file.
4. Read each question and enter your answer.
5. View your final score at the end of the quiz.
